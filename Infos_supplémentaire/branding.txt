# Mirakl Branding Guidelines

## Core Principles

- **Always** use Roboto Serif as the font family (titles, subtitles, body text)
- **Always** use the official color palette — no invented or approximated colors
- Aim for a **premium, trustworthy, minimalistic enterprise look**
- Use accent colors **strategically** to highlight key insights
- Follow **accessibility &amp; readability** best practices
- Do **not** overuse secondary colors — they support, never replace, primary colors

## Layout &amp; Components

### Cards
- `border-radius: 8px`
- Box shadow: `0 1px 4px rgba(0,0,0,0.1)` (subtle) or `0 2px 8px rgba(0,0,0,0.12)` (medium)
- Background options: `#03182F` (dark), `#F2F8FF` (light), `#FFFFFF` (white)
- Title: Roboto Serif, bold, `#03182F`

### Spacing Scale
| Token    | Value |
|----------|-------|
| xs       | 8px   |
| sm       | 16px  |
| md       | 24px  |
| lg       | 32px  |
| xl       | 48px  |
| xxl      | 64px  |
| generous | 96px  |

Use generous spacing between elements. Do not clutter. Whitespace creates structure.

---

## Applying the Guidelines

When generating any design artifact, apply the above tokens consistently:

1. **Set the font**: Import and use `Roboto Serif` from Google Fonts or declare `font-family: 'Roboto Serif', serif`.
2. **Build hierarchy**: Use H1–H5 rules strictly. Never invent font sizes.
3. **Color with intent**: Primary palette dominates. Accent blue (`#2764FF`) for KPIs and highlights. Secondary palette only for nuance.
4. **Space generously**: Use the spacing scale. Prefer `md` (24px) as default padding.
5. **Compose with cards**: Default card style is white background, 8px radius, subtle shadow.
6. **Check contrast**: All text must be legible. Dark text on light backgrounds, light text on dark backgrounds.
